# Product Overview

Bento Manager is a modern web application for managing bento (Japanese lunchbox) operations, built with Rails 8.1 and modern frontend tooling.

## Core Capabilities

- Bento management and operations
- Modern SPA-like experience with Hotwire (Turbo + Stimulus)
- Real-time updates and reactive UI
- Containerized deployment with Kamal
- Production-ready with Solid Queue, Solid Cache, and Solid Cable

## Target Use Cases

- Bento preparation and ordering workflow
- Inventory and schedule management
- Customer-facing interface for browsing and ordering
- Administrative backend for operations management

## Value Proposition

Combines the robustness of Rails with modern frontend interactivity, providing a fast, reliable solution for bento operations with minimal JavaScript complexity. Leverages Rails 8's modern stack (Solid gems) for simplified deployment and reduced infrastructure dependencies.
